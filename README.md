## How the Game Logic Works

- The computer randomly selects rock, paper, or scissors using Math.random()
- Math.floor() is used to convert the random number into an array index
- The user selects an option by clicking a button
- Both choices are compared using predefined win, lose, and draw conditions
- Scores and result messages are updated dynamically in the UI
